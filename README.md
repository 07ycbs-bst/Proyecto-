# Sistema de Gestión Fluye T&B

### Descripción
Aplicación web profesional diseñada para el control y gestión de servicios industriales. Este proyecto forma parte de mi formación en Ingeniería de Sistemas.

### Tecnologías utilizadas
* **Lenguaje:** PHP 8.x
* **Base de Datos:** MariaDB / MySQL (usando PDO para mayor seguridad)
* **Interfaz:** HTML5, CSS3 y Bootstrap
* **Herramientas:** cPanel, HeidiSQL y GitHub
